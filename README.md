# adidas-sales-dashboard
1. Project Title / Headline
Adidas Sales Analytics: Global Sales Performance Dashboard
A dynamic and interactive data visualization dashboard designed to analyze Adidas sales performance across regions, product categories, and time periods. The dashboard helps stakeholders identify revenue trends, top-performing products, and regional market performance.
2. Short Description / Purpose
The Adidas Sales Analytics Dashboard is a visually engaging and analytical Power BI report created to explore and evaluate Adidas sales data across different regions and product segments. The dashboard provides insights into key metrics such as total revenue, units sold, profit margins, and sales distribution.
It allows business analysts, marketing teams, and management to monitor sales performance, identify high-demand products, and analyze regional trends. By using interactive visuals and filters, users can easily explore sales patterns, compare performance across regions, and make data-driven decisions to improve business strategies and customer reach.
3. Tech Stack
The dashboard was built using the following tools and technologies:
Power BI Desktop
Main platform used to design the dashboard and create interactive data visualizations.
Power Query
Used for data cleaning, transformation, and preparation before loading data into Power BI.
DAX (Data Analysis Expressions)
Used to create calculated measures such as total sales, profit, profit margin, and year-to-date sales.
Data Modeling
Relationships were created between tables such as sales data, product information, retailer details, and regional data to support accurate analysis.
Excel / CSV Data Sources
Used as the primary dataset source containing Adidas sales records.
4. Data Source
The dataset used for this dashboard contains Adidas retail sales data including product categories, retailers, regions, and sales performance metrics.
Key fields in the dataset include:
Retailer Name
Region and State
Product Category
Units Sold
Price per Unit
Total Sales
Operating Profit
Operating Margin
Sales Date
This dataset helps analyze Adidas sales across different geographic regions and product lines.
5. Key Metrics (KPIs)
The dashboard highlights several important performance indicators:
Total Sales Revenue
The overall revenue generated from Adidas product sales.
Total Units Sold
The total number of Adidas products sold across all regions.
Operating Profit
Profit earned after deducting operating expenses from total revenue.
Operating Margin
Percentage of profit relative to total sales.
Average Price per Unit
The average selling price of Adidas products.
These KPIs provide a quick overview of Adidas business performance.
6. Dashboard Features
The Adidas Sales Dashboard includes several interactive visuals:
Sales by Region Map
Displays total sales distribution across different geographic regions.
Sales by Product Category
Shows which Adidas product categories generate the highest revenue.
Sales Trend Over Time
A line chart showing monthly or yearly sales trends.
Retailer Performance Analysis
Highlights the top retailers contributing to Adidas sales.
Profit Analysis
Shows operating profit and margin across regions and products.
Users can filter data by region, product category, retailer, and time period to explore specific insights.
